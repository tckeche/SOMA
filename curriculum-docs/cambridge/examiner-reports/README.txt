Cambridge Examiner Report Archive Manifest

What is included:
- IGCSE and A Level folders
- One subject folder per syllabus code
- For each year 2021-2025 and session March/June/November, a shortcut file named as CODE_Year_Session_ER.url
- A master CSV manifest listing the intended PDF filename and the public source URL

Important limitation:
This environment could verify the public URL naming pattern, but bulk-downloading hundreds of PDFs directly was not technically available here.
So this package gives you an exact, organized retrieval manifest and clickable shortcuts rather than the binary PDFs themselves.

Filename convention requested by user:
- CODE_Year_Session_ER.pdf

Source URL convention used:
- https://pastpapers.papacambridge.com/directories/CAIE/CAIE-pastpapers/upload/CODE_[m|s|w]YY_er.pdf
  where m=March, s=May/June, w=October/November

Examples verified during construction:
- 0610_m25_er.pdf
- 0610_s25_er.pdf
- 0475_m25_er.pdf
