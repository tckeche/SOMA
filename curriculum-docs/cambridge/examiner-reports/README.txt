Cambridge full 6-year archive (2021-2026)

Structure
- IGCSE/<Subject>/<Year>/...pdf
- A_Level/<Subject>/<Year>/...pdf

Important notes
1) Cambridge syllabus PDFs usually cover all exam sessions available for the year(s) named inside the PDF, so separate March/June/November session syllabus PDFs generally do not exist.
2) For some subjects, Cambridge's public site currently exposes one PDF covering multiple years rather than a separate PDF for each individual year. Those official multi-year PDFs have been copied into each relevant year folder.
3) Where an exact public PDF for a single requested year was not available from Cambridge's current public archive, the nearest official Cambridge public PDF has been placed in that year folder. In a few folders, two neighbouring official PDFs are included to bracket a missing year.
4) A Level Pure Maths, Mechanics and Statistics all use Cambridge International AS & A Level Mathematics (9709), because Cambridge treats these as routes/options inside one syllabus rather than separate syllabi.
5) At IGCSE level, Cambridge does not provide stand-alone long-run syllabi matching the labels 'Mechanics' and 'Statistics' in the same way. Those folders contain the closest Cambridge mathematics PDFs used as practical matches.
6) A Level French changes code across this period: older folders use 9716 as the nearest official match; 2025 onward uses 9898.
7) IGCSE Business Studies 0450 is used through 2026 in this archive because that is the requested 2021-2026 window.
